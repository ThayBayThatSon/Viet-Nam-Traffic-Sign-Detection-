# Vietnamese Traffic Sign Recognition Using YOLOv8

This project is a computer vision application that detects and recognizes Vietnamese traffic signs using the YOLOv8 object detection model. The project provides both a Web Application and a Desktop Application interface for easy interaction.

## 📂 Project Structure

- `app.py`: Main script to run the Web Application.
- `desktop_app.py`: Main script to run the Desktop GUI Application.
- `models/`: Contains the trained YOLOv8 model weights (e.g., `best_traffic_sign_model.pt`).
- `data/`: Datasets used for training and testing.
- `runs/`: Output directory generated by YOLOv8 containing training logs and validation metrics.
- `notebooks/` & `scripts/`: Jupyter notebooks and Python scripts for data processing, model training, and evaluation.
- `presentation/`: Contains presentation slides, HTML versions, scripts, and related resources.
- `templates/` & `static/`: HTML templates and static assets (CSS, JS, images) for the Web Application.
- `requirements.txt`: List of Python dependencies required to run the project.

## 🚀 Getting Started

### 1. Prerequisites

Ensure you have Python 3.8+ installed on your system. It is recommended to use a virtual environment.

### 2. Installation

Clone this repository and navigate to the project directory:

```bash
git clone <your-repository-url>
cd <repository-name>
```

Install the required dependencies using pip:

```bash
pip install -r requirements.txt
```

### 3. Running the Applications

#### Web Application

To start the web interface, run:

```bash
python app.py
```

Then open your web browser and navigate to the local address provided in the terminal (usually `http://localhost:5000` hoặc `http://127.0.0.1:5000`).

#### Desktop Application

To launch the desktop GUI, run:

```bash
python desktop_app.py
```

## 🧠 Model Training

If you wish to retrain or fine-tune the model, refer to the resources in the `scripts/` or `notebooks/` directory. Make sure your dataset is correctly formatted in the YOLO format and placed inside the `data/` folder.
